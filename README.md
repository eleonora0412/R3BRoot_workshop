# R3B Analysis & Simulation Course Repository

## Overview

This repository contains the material used in the R3B analysis and simulation course.
It provides example ROOT/FairRoot macros, configuration files, and documentation to help users learn how to:

- Analyze experimental data from the R3B experimental setup
- Run detector simulations 
- Perform detector calibrations
- Understand typical R3B analysis workflows

The repository is intended for students and early-career researchers, but it can also serve as a reference for more advanced users.
